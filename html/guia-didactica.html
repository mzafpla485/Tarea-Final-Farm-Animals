📘 Guía didáctica

This teaching guide helps teachers to implement the project "Farm Animals" in 1st grade of Primary Education.

🎯 Relationship between activities and evaluation criteria

Activity Description Evaluation criteria
Session 1: Introduction Discovering the animals by using flashcards Recognises basic vocabulary related to farm animals
Session 2: Vocabulary Learning and repeating animal names and its charcateristics Recognises basic vocabulary
Session 3: Song Listen to the song and sing it Associates words with images and sounds
Session 4: Grammar (production) Drawing and completing simple sentences Produces simple written/oral structures
Session 5: Guess the animal  Interactive games Associate words with images
Session 6&7: Final task Prepare and presentat your favourite farm animal

Participates in basic oral interaction using simple expressions

🧠 Key competences

  • Linguistic communication
  • Digital competence
  • Learning to learn

📝 Recommendations for teachers

  • Use visual support (flashcards, images)
  • Repeat structures frequently
  • Encourage participation in a safe environment
  • Adapt activities to different learning rhythms

🌈 Attention to diversity:

  • Visual support
    Simple language
    Flexible tasks

🧠 DUA principles:

  • Multiple ways of learning (audio, images, games)
    Different ways to express learning (drawing, speaking, digital)
    Motivation through play and creativity
